## User Story 1: Visualizar Refeição Aleatória
Como usuário
Quero ver uma refeição aleatória
Para que eu possa descobrir novas opções de pratos para preparar

Critérios de Aceite:
- Deve exibir o nome da refeição
- Deve mostrar uma imagem do prato
- Deve exibir a lista de ingredientes necessários
- Deve exibir as instruções de preparo
- Deve carregar uma nova refeição aleatória a cada requisição

## User Story 2: Atualizar Refeição Aleatória
Como usuário
Quero poder gerar uma nova refeição aleatória
Para que eu possa ver outras opções se a atual não me agradar

Critérios de Aceite:
- Deve ter um botão "Nova Refeição"
- Deve carregar uma refeição diferente ao clicar no botão
- Deve mostrar um indicador de carregamento durante a requisição
- Não deve repetir a refeição anterior imediatamente

## User Story 3: Visualizar Detalhes Nutricionais
Como usuário
Quero ver informações nutricionais da refeição
Para que eu possa avaliar se ela atende minhas necessidades dietéticas

Critérios de Aceite:
- Deve exibir as calorias totais da refeição
- Deve mostrar a quantidade de proteínas, carboidratos e gorduras
- Deve indicar se a refeição é vegetariana, vegana ou sem glúten
- Deve apresentar o tempo estimado de preparo
- As informações nutricionais devem ser exibidas de forma clara e organizada
